# Портфолио

| Название папки  | Название проекта | Описание проекта |
| -------------   | -------------    | -------------    |
| 1_game_industry_EDA   | Исследование продаж компьютерных игр     | Выявление закономерностей, определяющих успешность игр. Предоставлены исторические данные о продажах игр в различных регионах.  Предобработка и EDA исторических данных о продажах компьютерных игр. Построение портрета пользователя каждого региона. Проверка гипотез.    |
| 2_tariff_recomendation    | Рекомендация тарифов    | Построение модели МО, предсказывающей наиболее подходящий тариф для клиента мобильного оператора. Предоставлены данные о пользовании тарифами клиентами оператора. Задача бинарной классификации. Метрика - accuracy. Исследование эффективности множества различных моделей МО.     |
| 3_customer_churn    | Отток клиентов     | Построение модели МО, предсказывающей возможный уход человека из клиентов банка. Предоставлены исторические данные о поведении клиентов и расторжении договоров с банком. Задача бинарной классификации. Метрика - AUC-ROC и F1-мера. Борьба с дисбалансом классов.     |
| 4_oil_well_selection_bootstrap    | Выбор локации для скважины     | Построение модели МО для предсказания прибыльности нефтяных скважин. Предоставлены синтетические данные о существующих нефтяных скважинах в 3х различных регионах. Задача регрессии. Метрика - RMSE. Применение техники Bootstrap для определения наиболее прибыльных регионов.     |
| 5_gold_recovery    | Восстановление золота из руды     | Построение модели МО для предсказания коэффициента восстановления золота из золотосодержащей руды. Предоставлены данные  о различных химических и физических показателях во время этапов флотации и очистки руды. Задача регрессии. Метрика - sMAPE. Построение 2х разных моделей для 2х этапов очистки и восстановления золота из руды.    |
| 6_protection_of_personal_data    | Защита персональных данных клиентов     | Разработка метода преобразования данных, по которому будет сложно восстановить персональную информацию. Предоставлены персональные данные клиентов страховой компании. Задача линейной алгебры. Создание и тестирование алгоритма преобразования данных на основе умножении матриц и поиске обратной матрицы.     |
| 7_autos_cost_determination    | Определение стоимости автомобилей     | Построение модели МО, предсказывающей стоимость автомобиля. Предоставлены данные о технических и эксплуатационных характеристиках б/у автомобилей и их стоимости. Задача регрессии. Метрика - RMSE. Определение не только наиболее качественной, но и самой быстрой модели МО.     |
| 8_taxi_time_series    | Прогнозирование заказов такси     | Построение модели МО для предсказания количества заказов такси на следующий час. Предоставлены исторические данные о заказах такси во временном ряде. Задача регрессии. Метрика - RMSE. Анализ временных рядов и feature engineering для построения эффективной модели МО.    |
| 9_text_sentiment_analysis   | Анализ токсичности комментариев     | Построение модели МО для выявления токсичных комментариев. Предоставлена база комментариев с метками о токсичности. Задача классификации. Метрика - F1-мера. Feature engineering, лемматизация и очистка текста, TF-IDF векторизатор и проба модели BERT в качестве альтернативы.     |
